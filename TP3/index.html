Le langage SQL

1.1. Définition

Le langage SQL (Structured Query Language ou Langage de requêtes structuré) est un langage comprenant l’ensemble des ordres nécessaires à la création et à la gestion d’une base de données relationnelle.

Il a été créé en 1986 et est l’aboutissement de plusieurs langages créés depuis l’apparition du modèle relationnel en 1970 (inventé par Mr Codd, mathématicien chez IBM).

L'utilisation du langage SQL suppose que les données soient définies selon le modèle relationnel au sein d'un SGBDR (Oracle, SQL Server, MySQL, PostgreSQL…).

1.2. Les différentes normes utilisées

Il existe plusieurs normes SQL :

  • SQL 89
  • SQL 92
  • SQL 2011

Malgré les normes, il peut exister des différences entre les SGBDR. Voici un extrait du site PostgreSQL :

"Le développement de PostgreSQL respecte le standard en vigueur, tant que celui-ci ne s'oppose pas aux fonctionnalités traditionnelles ou au bon sens…"

"SQL-92 définit trois niveaux de conformité : ENTRY, INTERMEDIATE et FULL."
