1. INTRODUCTION AU CSS

CSS signifie Cascading Style Sheets (Feuilles de style en cascade). CSS est utilisé pour mettre en forme et présenter les éléments HTML d'une page web.

Tandis que HTML fournit la structure, CSS fournit le style et la mise en page.

2. SYNTAXE CSS

La syntaxe CSS est basée sur des règles composées d'un sélecteur et d'une déclaration :

selecteur {
    propriete: valeur;
    propriete2: valeur2;
}

Exemple :

h1 {
    color: blue;
    font-size: 2rem;
}

3. LES SÉLECTEURS

Les sélecteurs CSS permettent de cibler les éléments HTML à mettre en forme :

Sélecteur d'élément

p {
    color: black;
}

Sélecteur de classe

.ma-classe {
    background-color: yellow;
}

Sélecteur d'ID

#mon-id {
    font-weight: bold;
}

Sélecteur universel

* {
    margin: 0;
    padding: 0;
}

4. PROPRIÉTÉS COURANTES

Voici les propriétés CSS les plus utilisées :

Couleurs et fond

  • color : couleur du texte
  • background-color : couleur de fond
  • background-image : image de fond

Texte

  • font-size : taille du texte
  • font-weight : épaisseur du texte
  • text-align : alignement du texte

Boîte (Box Model)

  • margin : marge externe
  • padding : marge interne
  • border : bordure
  • width : largeur
  • height : hauteur
