1. INTRODUCTION AU HTML

HTML signifie HyperText Markup Language. C'est le langage de balisage utilisé pour créer des pages web. HTML fournit la structure et le contenu d'une page web.

Contrairement à un langage de programmation, HTML n'est pas un langage de programmation à proprement parler, mais un langage de balisage qui utilise des balises pour structurer le contenu.

2. STRUCTURE D'UNE PAGE HTML

Toute page HTML doit respecter une structure de base :

<!DOCTYPE html>
<html lang="fr">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Titre de la page</title>
        <link rel="stylesheet" href="style.css">
    </head>
    <body>
        <!-- Contenu de la page -->
    </body>
</html>

Chaque élément a un rôle spécifique :

  • <!DOCTYPE html> : déclare le type de document
  • <html> : élément racine
  • <head> : contient les métadonnées
  • <body> : contient le contenu visible

3. LES BALISES PRINCIPALES

Voici les balises les plus couramment utilisées en HTML :

Titres

<h1>Titre principal</h1>
<h2>Titre secondaire</h2>
<h3>Titre tertiaire</h3>

Paragraphes et texte

<p>Ceci est un paragraphe</p>
<strong>Texte en gras</strong>
<em>Texte en italique</em>

Listes

<ul>
    <li>Élément 1</li>
    <li>Élément 2</li>
</ul>

Liens et images

<a href="https://exemple.com">Lien</a>
<img src="image.png" alt="Description">
