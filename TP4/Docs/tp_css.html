OBJECTIF GÉNÉRAL

Ce TP a pour objectif de vous familiariser avec les sélecteurs CSS et les propriétés de mise en forme.

Vous apprendrez à :

  • Utiliser les différents types de sélecteurs (élément, classe, ID)
  • Appliquer des styles de couleur et de fond
  • Mettre en forme le texte
  • Gérer les marges et les espacements
  • Créer des mises en page avec Flexbox

EXERCICES À RÉALISER

Exercice 1 : Mise en forme basique

Prenez une page HTML simple et appliquez du CSS pour :

  • Changer les couleurs des titres
  • Modifier la police et la taille du texte
  • Ajouter un fond coloré
  • Ajuster les marges et les espacements

Exercice 2 : Sélecteurs avancés

Créez une page avec plusieurs éléments et utilisez :

  • Des sélecteurs de classe pour grouper des styles
  • Des sélecteurs d'ID pour des éléments uniques
  • Des sélecteurs combinés (enfant, frère, etc.)

Exercice 3 : Mise en page avec Flexbox

Créez une mise en page responsive avec :

  • Un en-tête flexible
  • Une barre latérale et un contenu principal
  • Un pied de page qui s'adapte à la largeur de l'écran
  • Des éléments alignés correctement avec Flexbox
