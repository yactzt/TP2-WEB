OBJECTIF GÉNÉRAL

Ce TP a pour objectif de vous familiariser avec la structure de base d'une page HTML et l'utilisation des balises principales.

Vous apprendrez à :

  • Créer une page HTML valide
  • Utiliser les balises de structure (<header>, <main>, <footer>)
  • Organiser le contenu avec des titres et des paragraphes
  • Créer des listes et des liens
  • Insérer des images

EXERCICES À RÉALISER

Exercice 1 : Page personnelle

Créez une page HTML présentant votre profil avec :

  • Un titre principal
  • Une photo de profil
  • Une description personnelle
  • Une liste de compétences
  • Des liens vers vos réseaux sociaux

Exercice 2 : Page de blog

Créez une page de blog avec :

  • Un en-tête avec le titre du blog
  • Au moins 3 articles avec titre, date et contenu
  • Un pied de page avec les informations de contact

Exercice 3 : Page de portfolio

Créez une page de portfolio présentant vos projets :

  • Une galerie de projets avec images
  • Une description pour chaque projet
  • Des liens vers les projets en ligne
